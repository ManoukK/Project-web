Dit is de website voor het vliegtuig maatschappij WOW air. Deze website krijgen klanten die bij WOW air een vliegticket hebben gekocht. Zij krijgen 99 uniek vormgegeven verhalen meer voor op reis.

De website is responsive en multi device zodat iedereen de website kan gebruiken op het apparaat naar keuze.

Voor de gebruiker is het belangrijk dat:
- de verhalen offline te lezen zijn omdat in een vliegtuig vaak geen internet aanwezig is.
- de gebruiker leuke verhalen kan vinden.
- de opgeslagen verhalen makkelijk en snel terug te vinden zijn.
- de gebruiker weet hoe de verhalen bij andere in de smaak vallen.

Deze doelen heb ik terug laten komen op de website door:
- een download knop toe te voegen aan alle verhalen.
- veel opties mee te geven om een leuk verhaal te vinden.
- op de homepage gelijk naar de opgeslagen te kunnen.
- de optie mee te geven om verhalen te reviewen en door een lijst op de homepage te laten zien wat de populaire verhalen zijn van deze week.
